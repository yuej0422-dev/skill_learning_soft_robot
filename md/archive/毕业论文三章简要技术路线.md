# 毕业论文三章简要技术路线

## 一、总体研究主线

本文围绕软体机械臂从**状态空间运动控制**到**视觉状态感知**，再到**视觉—语言驱动的复杂任务执行**展开，形成以下递进关系：

\[
\boxed{
\text{特权状态下的目标条件控制}
\rightarrow
\text{无动捕条件下的视觉状态恢复}
\rightarrow
\text{面向精细操作的 VLA 后训练}
}
\]

第一章提供可靠的底层运动控制能力；第二章将动捕状态转化为仅在训练阶段使用的特权监督信息，建立视觉观测到状态及 Koopman 潜在空间的映射；第三章在前两章基础上对 VLA 进行监督微调和强化学习后训练，实现语义理解、未来状态预测与精细动作生成。

---

## 二、第一章：基于 Koopman 状态空间的目标条件强化学习控制

第一章基于动捕系统获得由 TCP 位姿与速度组成的 12 维状态，并通过 Koopman 编码器将非线性状态提升至近似线性演化的潜在空间。在此基础上构建目标条件强化学习控制策略，使同一控制器能够根据当前状态与目标状态生成连续控制动作，实现软体机械臂在不同目标条件下的稳定运动控制。

---

## 三、第二章：基于 RGB-D 三维重构与 Koopman 对齐的视觉状态表征

第二章面向实际部署时无法持续获取动捕状态的问题，使用单台 RGB-D 相机采集连续多帧观测。首先通过深度反投影获得软体机械臂的三维点云，并利用三维几何编码器提取逐帧构型特征；随后结合多帧几何特征和历史动作，通过时序网络估计包含 TCP 位姿与速度的 12 维状态。

训练阶段以动捕状态作为绝对正确的特权监督，将视觉潜变量对齐至第一章已经学习的 Koopman 潜在空间，并施加 Koopman 演化一致性和控制输出一致性约束。部署阶段移除动捕输入，仅由 RGB-D 历史观测产生视觉状态和 Koopman latent，继续驱动第一章的目标条件控制器。

核心流程为：

\[
I_{t-K+1:t}^{RGBD}
\rightarrow
P_{t-K+1:t}^{3D}
\rightarrow
E_{\mathrm{3D}}
\rightarrow
E_{\mathrm{temp}}
\rightarrow
\{\hat s_t,\,z_t^V\},
\]

其中：

\[
\hat s_t\in\mathbb R^{12},
\qquad
z_t^V\approx E_s(s_t^{\mathrm{mocap}}).
\]

主要训练目标为：

\[
\mathcal L_{\mathrm{chapter2}}
=
\lambda_s\mathcal L_{\mathrm{state}}
+
\lambda_a\mathcal L_{\mathrm{align}}
+
\lambda_k\mathcal L_{\mathrm{koop}}
+
\lambda_c\mathcal L_{\mathrm{control}}.
\]

必要时可加入轻量三维几何重构损失，以保证视觉特征保留软体机械臂的整体三维构型。

---

## 四、第三章：基于 Koopman 未来先验与紧凑潜在状态的 VLA 后训练

### 4.1 研究目标

第三章主要解决预训练 VLA 在软体机械臂部署中的两类问题：

1. VLA 的视觉—语言主干主要继承静态图文预训练知识，缺少对软体机械臂连续动态、迟滞及动作后果的显式建模；
2. 直接对高维 VLA embedding 或完整大模型进行强化学习微调，计算成本高、样本效率低，且真实机器人在线训练不稳定。

因此，第三章采用以下整体路线：

\[
\boxed{
\text{基础 SFT}
\rightarrow
\text{Koopman 未来先验增强}
\rightarrow
\text{紧凑 latent 提取}
\rightarrow
\text{轻量在线 RL 精修}
}
\]

### 4.3 Koopman 引导的 Latent Steering

参考 *World Pilot* 中 Latent Steering 的思想，不直接生成未来 RGB 图像，而是向 VLA 感知层注入紧凑的未来场景演化 latent。

本研究使用第二章得到的视觉 Koopman 状态作为当前动态状态：

\[
z_t^V.
\]

训练阶段可使用专家动作 chunk，推理阶段使用基础 VLA 初步生成的参考动作 chunk，在第一章固定的 Koopman 动力学中进行多步 rollout：

\[
\tilde z_{t+h}
=
A^h z_t^V
+
\sum_{j=0}^{h-1}
A^{h-1-j}B a_{t+j}^{prior},
\qquad
h=1,\ldots,H.
\]

其中：

\[
a_{t:t+H}^{prior}
=
\begin{cases}
a_{t:t+H}^{*}, & \text{训练阶段的 teacher forcing},\\
a_{t:t+H}^{ref}, & \text{推理或闭环训练阶段}.
\end{cases}
\]

将未来 latent 序列编码为场景演化先验：

\[
Z_t^{future}
=
E_{\mathrm{future}}
\left(
\tilde z_{t+1:t+H}
\right).
\]

随后通过残差交叉注意力将其注入 VLA 的视觉—语言 hidden states：

\[
H_t^{steer}
=
H_t^{VLM}
+
\operatorname{CrossAttn}
\left(
Q=H_t^{VLM},
K=Z_t^{future},
V=Z_t^{future}
\right).
\]

该模块使 VLA 在生成动作前，不仅理解当前图像和语言语义，还能够感知当前动作可能导致的未来 TCP 状态变化、运动趋势及动作后果。与直接预测未来图像相比，Koopman future latent 更紧凑，并且与第一、二章的状态空间和动力学模型保持统一。

### 4.4 Koopman 引导的 Action Steering

参考 *World Pilot* 中 Action Steering 的思想，为最终动作生成器提供轨迹级运动先验，而不是直接用先验轨迹替换 VLA 输出。

首先由基础 VLA 产生初步参考动作：

\[
a_{t:t+H}^{ref}.
\]

该动作在 Koopman 空间中产生未来状态轨迹：

\[
\tilde z_{t+1:t+H}
=
\operatorname{Rollout}_{A,B}
\left(
z_t^V,a_{t:t+H}^{ref}
\right).
\]

随后根据参考动作及其对应的未来状态轨迹，编码得到轨迹级 motion-prior token：

\[
p_t^{act}
=
E_{\mathrm{act}}
\left(
a_{t:t+H}^{ref},
\tilde z_{t+1:t+H},
g_t
\right).
\]

其中 \(g_t\) 可由明确目标状态给出，也可由 VLA 根据视觉和语言任务隐式生成。

将该 token 作为 prefix 输入 action generator：

\[
a_{t:t+H}^{final}
=
G_{\mathrm{action}}
\left(
H_t^{steer},
z_t^V,
p_t^{act}
\right).
\]

Action Steering 不要求最终动作逐点复制参考轨迹，而是提供整体运动方向、速度变化和未来状态演化趋势，使动作生成器在保留生成自由度的同时，减少动力学不一致或偏离目标的动作。

因此，第三章前半部分形成两条互补路径：

\[
\boxed{
\text{Latent Steering：在感知层注入未来状态演化先验}
}
\]

\[
\boxed{
\text{Action Steering：在动作层注入轨迹级运动先验}
}
\]

### 4.5 面向强化学习的紧凑 latent state 提取

在完成监督微调和 Koopman Steering 后，进一步参考 RL Token（RLT）的思想，从 VLA 的高维 token embedding 中提取一个紧凑、任务相关的 latent state，作为在线强化学习接口。

设 VLA 输出的高维 token 序列为：

\[
H_t=
[h_t^1,h_t^2,\ldots,h_t^L]
\in\mathbb R^{L\times d}.
\]

在序列中加入可学习的特殊 token，并通过轻量 Transformer encoder 提取紧凑状态：

\[
z_t^{RL}
=
E_{\mathrm{RLT}}
\left(
H_t,q_{\mathrm{RL}}
\right).
\]

随后使用 decoder 从 \(z_t^{RL}\) 重构冻结的 VLA embedding：

\[
\hat H_t
=
D_{\mathrm{RLT}}
\left(
z_t^{RL}
\right),
\]

并优化：

\[
\mathcal L_{\mathrm{RLT}}
=
\left\|
\hat H_t-
\operatorname{sg}(H_t)
\right\|^2.
\]

该瓶颈重构任务使 \(z_t^{RL}\) 尽可能保留高维 VLA 表征中与视觉、语言、任务阶段和动作生成相关的信息，同时将强化学习输入压缩到较低维度。

结合第二章的视觉动态状态，在线 RL 状态可构造为：

\[
x_t^{RL}
=
[z_t^{RL},z_t^V],
\]

如有必要，还可加入少量可直接读取的驱动器状态或上一动作。

### 4.6 基于参考动作残差的在线强化学习精修

在线 RL 阶段冻结：

- VLA 主干；
- Koopman 状态编码与动力学模型；
- 已训练稳定的 Latent Steering 与 Action Steering 模块；
- RLT encoder。

基础 VLA 输出参考动作 chunk：

\[
a_{t:t+H}^{ref}
=
\pi_{\mathrm{VLA}}
(I_t,l_t,z_t^V).
\]

轻量 actor 不从零生成动作，而是输出参考动作的残差：

\[
\Delta a_{t:t+H}
=
\pi_{\mathrm{actor}}
\left(
x_t^{RL},a_{t:t+H}^{ref}
\right).
\]

最终执行：

\[
a_{t:t+H}
=
\operatorname{clip}
\left(
a_{t:t+H}^{ref}
+
\Delta a_{t:t+H}
\right).
\]

critic 评估紧凑 latent state 下动作 chunk 的价值：

\[
Q
\left(
x_t^{RL},a_{t:t+H}
\right).
\]

actor 的训练目标同时考虑价值最大化和参考动作约束：

\[
\mathcal L_{\mathrm{actor}}
=
-
Q
\left(
x_t^{RL},a_{t:t+H}
\right)
+
\lambda_{\mathrm{anchor}}
\left\|
a_{t:t+H}-a_{t:t+H}^{ref}
\right\|^2.
\]

该设计使在线 RL 主要在 VLA 已有行为附近进行局部精修，而不是在高维动作空间中从零探索，从而：

1. 冻结大规模 VLA，仅训练小型 actor–critic，提高微调效率；
2. 使用紧凑 latent state 降低在线强化学习的状态维度；
3. 保留预训练 VLA 的视觉语义理解和任务泛化能力；
4. 提升插入、对准、接触等精细操作阶段的成功率；
5. 利用 Koopman future latent 和动作先验，使 RL 精修尽量保持动力学可行性。

### 4.7 第三章训练流程

#### 阶段一：VLA 监督微调

训练基础动作生成能力：

\[
(I_t,l_t,z_t^V)
\rightarrow
a_{t:t+H}^{*}.
\]

#### 阶段二：Koopman Steering 后训练

加入：

- Koopman future latent；
- Latent Steering；
- Koopman-consistent motion-prior token；
- Action Steering。

继续优化最终动作 chunk 的监督损失，使模型学会利用未来状态演化和动作轨迹先验。

#### 阶段三：RLT 表征训练

冻结或基本冻结 VLA，通过 encoder–decoder bottleneck 从高维 embedding 中提取：

\[
z_t^{RL}.
\]

#### 阶段四：在线强化学习精修

冻结大模型和 latent encoder，训练轻量 actor–critic，使 actor 围绕 VLA 参考动作输出残差 action chunk，重点提升软体机械臂在接触丰富和高精度阶段的性能。

## 五、三章之间的统一关系

三章之间的数据和模型接口为：

\[
s_t^{\mathrm{mocap}}
\xrightarrow{\text{第一章}}
z_t^S,\;A,\;B,\;\pi_{\mathrm{GC}},
\]

\[
I_{t-K+1:t}^{RGBD}
\xrightarrow{\text{第二章}}
\hat s_t,\;z_t^V,
\]

\[
(I_t,l_t,z_t^V,A,B)
\xrightarrow{\text{第三章}}
a_{t:t+H}^{final}.
\]

其中：

- 第一章提供基础动力学表示和目标条件运动控制能力；
- 第二章使系统摆脱部署阶段对动捕状态的依赖；
- 第三章利用 Koopman 未来先验增强 VLA 的动态理解和动作生成，并通过紧凑 latent state 与轻量在线 RL 提升微调效率和精细操作能力。

最终系统形成：

```text
单 RGB-D 多帧观测
        ↓
三维视觉状态与 Koopman latent
        ↓
视觉 + 语言 + 当前动态状态
        ↓
Koopman Latent Steering
        ↓
Koopman Action Steering
        ↓
SFT 后的 VLA 参考动作 chunk
        ↓
RLT 紧凑 latent + 轻量 actor–critic
        ↓
残差修正后的最终动作 chunk
        ↓
软体机械臂精细操作
```

## 六、三章创新点概括

### 创新点一：目标条件 Koopman 强化学习控制

面向软体机械臂强非线性运动控制问题，构建基于 Koopman 状态空间的目标条件强化学习控制策略，实现不同目标条件下的统一连续运动控制。

### 创新点二：三维视觉 Koopman 状态表征

面向部署阶段状态不可直接获取的问题，利用 RGB-D 三维几何、多帧时序观测和动捕特权监督，学习与第一章 Koopman 空间一致的视觉动态状态，实现无动捕条件下的闭环状态感知。

### 创新点三：Koopman 未来先验与紧凑 latent 驱动的 VLA 后训练

面向 VLA 缺乏连续动力学认知以及整模型 RL 微调效率较低的问题，将 Koopman 预测的未来状态演化和动作轨迹分别注入感知层与动作生成层，并从高维 VLA embedding 中提取紧凑 RL latent state，通过轻量 actor–critic 对参考 action chunk 进行残差精修，从而提高 VLA 在软体机械臂精细操作中的适应效率、动作精度和任务成功率。
